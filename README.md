# SimpleReceipt

Clone repo
Make sure you have admin rights

Backend project:
  Open project
  Set api project as startup
  Configure connection string in appSettings.json
  Start
  http://localhost:65389/api/index.html
  
Frontend project:
  Open project
  Run "npm install" to setup dependencies
  Run "ng serve"
  http://localhost:4200/#/
